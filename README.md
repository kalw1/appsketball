# appsketball

A small app used to track stats of both teams during a basketball game.

## Running app

The app can either be run on your machine through an emulator, or via the APK.

### Android

1. Download the `appsketball-[DATE].apk` APK file to your Android device
2. In Settings, turn on 'Install Unknown Apps'
3. Install and run the APK

### VSCode

Prerequisites:
- VSCode
- Java JDK
- Gradle

1. Follow the Flutter install steps: https://docs.flutter.dev/install/quick
2. (OPTIONAL) Download Android Studio: https://developer.android.com/studio
3. In VSCode, choose the device to run on in Flutter Device
4. Run the app

If you want to generate an APK of your local changes (or the APK in the repository is out of date), run `flutter build apk` in terminal.
